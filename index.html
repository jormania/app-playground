Codex Alchymicus — KCD II
Kingdom Come: Deliverance II · Alchemy Recipe Manager
v3
HTML / CSS / JS Gaming Reference Tool Mobile Responsive WCAG AA No Dependencies

A medieval-styled interactive codex for tracking alchemy recipes in KCD II. Parchment-and-candlelight aesthetic, split-panel layout: browsable recipe index with category filtering on the left, detailed step-by-step preparation view on the right.

  • 28 pre-loaded recipes across 7 categories — Potions, Decoctions, Poisons, Perfumes, Powder, Special
  • Animated SVG candle flames, parchment texture, corner ornaments
  • Herb icons drawn in inline SVG (nettle, belladonna, marigold, poppy, chamomile…)
  • Category tabs, ingredient search, difficulty pips, Mastered toggle
  • Custom recipe modal with ingredient and step parsing
  • Full WCAG AA contrast compliance — v3 overhaul fixed all dark-panel failures
