Touch Grass v5
// feature guide

Everything Touch Grass does.

A complete reference for the app. The app itself is intentionally minimal — this page is where the details live.

// how to use

  1. Tap ↺ to pull a nearby green space, or type any location.
  2. Tap Head outside — your departure is saved immediately.
  3. Close the tab, lock your screen, walk. The timer runs in localStorage.
  4. Come back and open the app. Tap I'm back.
  5. See what the walk generated. Check your collection over time.
location

Live nearby places

On load, the app fetches your GPS position and queries OpenStreetMap's Overpass API for parks, forests, gardens, and nature reserves within 6 km. Results include named way, node, and relation types — so large forests mapped as polygon relations (like Vlăsia) are included.

Tap ↺ to pick a random place from the list. The list is cached per session so subsequent taps are instant. If GPS is unavailable, the query falls back to central Bucharest coordinates.

finds

Surreal discoveries

When you return, Claude generates discoveries scaled to your walk duration — roughly 2 finds per 15 minutes, capped at 15. The prompt specifies location, time of day, and current weather, and instructs Claude to generate dreamlike, impossible objects — never realistic litter.

Four rarity tiers, each with a defined aesthetic:

  • common Mildly uncanny. Something real that feels slightly wrong.
  • uncommon Clearly impossible. Physical laws politely ignored.
  • rare Reality-bending. The ground doesn't behave as expected.
  • legendary Mythic, awe-inspiring. 1% drop rate. When it happens, you'll know.
audio

Synthetic ambient soundscape

All audio is generated in real time via the Web Audio API — no samples, no files. It plays when you're on the "You're outside" screen and fades on return. Toggle with the speaker icon in the welcome screen header; preference is saved across sessions.

  • Wind — always present, intensity varies with weather. Storm mode raises it significantly.
  • Birds — dawn, morning, day, dusk. More frequent in parks and forests, reduced in urban locations.
  • Crickets — evening and night. Reduced in urban locations.
  • Rain — layered bandpass noise, triggered when weather is rain or storm.
  • Thunder — rare random bursts during storm conditions.
weather

Real-time weather integration

Current conditions are fetched from Open-Meteo using your GPS on every app load. No API key required. Weather affects three layers simultaneously:

  • Sky gradient — the colour palette shifts for clear, cloudy, storm, and fog conditions.
  • Canvas particles — rain streaks and snow dots are drawn directly on the canvas, tuned to intensity.
  • Ambient mix — rain layer volume and wind intensity track the weather state in real time.
sky

Celestial body

A sun or moon appears in the upper-right sky — never both. The sun is shown at dawn, morning, day, and dusk with a warm or golden tint matching the time. The moon appears at evening and night, drawn with the actual current lunar phase using a canvas clip technique.

Clouds drift in front of the celestial body at partial opacity for cloudy and snow conditions. The body is hidden entirely for rain and storm. Barely visible through fog.

stats

Stats row, streak & longest walk

Three icon boxes sit below the "Head outside" button on the welcome screen: 🔥 streak, 👣 total walks, 🎁 collection size. The streak box dims to 38% opacity when no active streak. Tap any box to open the full stats screen.

The stats screen breaks walks down by rolling 7-day week, current calendar month, current year, and all time. Your personal longest walk appears below the table.

The streak counter increments for each consecutive calendar day you complete at least one walk. Miss a day and it resets. Sessions are stored per-device — if you use both a laptop and a phone, track from one device for accurate counts.

history

Collection log

Every find from every walk is stored in localStorage, grouped by rarity tier. Stored finds are capped at 500 entries. Access via the 🎁 box on the welcome screen or the link on the results screen.

setup

API key & settings

Tap ⚙ on the welcome screen to set your Anthropic API key. It's stored in localStorage and sent directly to Anthropic's API from your browser — nothing passes through a server. A $5 credit covers several thousand walks. The app uses claude-sonnet-4-6.

install

Installing as an app (Android)

Touch Grass is a Progressive Web App. Open it in Chrome on Android, tap the three-dot menu, choose Add to Home Screen, then Install app (not just "Add shortcut"). It launches full-screen with no browser bar and the 🌿 icon. Works with Minimalist Phone and similar launchers.

Requires manifest.json, sw.js, and icon files to be present in the repo root — all are included in the app-playground repo.
